# Student Data Management System

## Overview

A menu-driven Student Data Management System developed using Java and File Handling. The application stores student records in a text file and supports complete CRUD operations.

## Features

* Add Student
* Display All Students
* Search Student by ID
* Update Student Details
* Delete Student Records
* Menu-Driven Interface
* Exception Handling

## Technologies Used

* Java
* Object-Oriented Programming (OOP)
* File Handling
* Exception Handling
